Post注入

POST注入一般分为两种，第一种绕过密码验证，第二种注入sql语句。

绕过密码验证

1’ or 1=1 #

由于sql语句中，or运算符的优先级是大于and我们发现１or 1=1 我们可以得知，1=1恒为真，or两侧只有有一个为真结果即为真。而#则直接注释掉了后面的sql语句，所以我们就能够直接绕过密码 直接进行登录。
