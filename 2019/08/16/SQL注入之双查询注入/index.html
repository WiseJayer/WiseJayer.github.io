双查询注入简介

双查询注入简单而言，就是一个select语句中还有一个select语句

双查询注入基本函数

rand()函数

rand函数是一个随机函数，它的作用是可以产生0~1之间的任意随机数。

floor()函数

floor函数是一个取整函数，结果返回小于等于你输入的数的整数。只返回数值的整数部分，小数部分忽略

concat（）函数

concat是一个连接函数，可以将两个字符串拼接为一个字符串。

count(*)函数

count(*)是一个统计行的函数，可以计算出由select语句返回的结果集中的行数。
