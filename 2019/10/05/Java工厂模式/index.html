工厂模式

非工厂模式:一个接口和一个子类绑定在了一起产生了耦合，A-B绑定在一起如果要修改的同时修改

工厂模式：A-B-C在A-B直接加多一个过渡类C，修改A是BC不影响，可移植性提高了，增加子类只要修改过渡类，避免了耦合问题
