多态

多态就是同一个接口，使用不同的实例而执行不同操作

  • 向上转型：父类 父类实例 = 子类实例（自动完成）
    • Father f = new Son();
  • 向下转型：子类 子类实例 = (子类)父类实例（强制完成转换）
    • Son s = (Son)f
