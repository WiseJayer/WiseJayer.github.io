泛型

类中的属性或者方法的参数与返回值可以由对象实例化时决定。

  • 泛型类
  • 泛型接口
  • 泛型方法

通配符<?>

不能使用在类上或者new创建对象上

上限

？ extends 类：设置泛型上限

表示泛型只允许设置该类或该类的子类

下限

？super 类：设置泛型的下限

表示泛型只允许使用该类或该类的父类
