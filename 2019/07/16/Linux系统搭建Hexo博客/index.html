前言

需要准备的东西：Github账号、Node.js环境、git工具

安装 Git

sudo yum install git-core

Node.js环境

下载 Node.js
wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
更新
source ~/.bash_profile
安装 Node.js
nvm install stable

安装 Hexo

用命令创建 hexo 文件夹
mkdir hexo
用命令安装 hexo
npm install -g hexo-cli
进入Hexo
cd hexo/
初始化 hexo
hexo init

测试

打开 hexo 服务
hexo server

部署到GitHub

设置 user.name 和 user.email
git config --global user.name "github名称"
git config --global user.email "github邮箱"
生成 ssh 密匙
ssh-keygen -t rsa -C "github邮箱"

然后出现密钥保存的路径一直按回车

查看密钥
cd .ssh
vim id_rsa.pub

复制里面的内容

:q

退出

创建 GitHub 仓库
命名格式为"github名称".github.io
添加密钥

登陆github—进入设置—找到ssh设置密钥把刚刚复制的粘贴进去即可

修改配置文件

打开 hexo 配置文件 _config.yml修改对应部分

cd hexo
vim _config.yml

找到以下内容并修改

deploy:
type: git
repo: git@github.com:github名称/github名称.github.io.git 
branch: master
生成并部署
hexo clean
hexo g
npm install hexo-deployer-git --save
hexo d

浏览

github名称.github.io
